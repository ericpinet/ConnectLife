ConnectLife

ConnectLife is a home automation system in java.

  • Coreserver: Main system server install on device without screen. (Javadoc API)
  • Simulator: Simulator of the home device (Helpful for tests).
  • Clapi: Client/Server protocol for the Connectlife system. (Javadoc API)
